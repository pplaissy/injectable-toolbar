Component injection with MVVM events

This application implements a way of injecting specific commands into a toolbar according to the context of the current view.

I'm using the MVVM event management library detailed in this previous article.

The main toolbar contains a menu of 3 pages, Home, Document, Charts.

When you display the Document page, justification commands are injected into the toolbar, which you can use to format the text.

When you display the Charts page, the commands injected into the toolbar offer different types of graph to render the datas.
